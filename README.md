> ## Features

> - Login, Register & Magic login link (via Email)
> - Dashboard admin

> - Fine system
> - QR Code for loan

>
> - [CodeIgniter 4](https://codeigniter.com/)
> - [CodeIgniter Shield](https://codeigniter4.github.io/shield/)

## How to Use

### PersyaConditionratan

- [Composer](https://getcomposer.org/).
- PHP 8.1+ and MySQL or [XAMPP](https://www.apachefriends.org/download.html) version 8.1+  by enabling the extension `-intl` and `-gd`.
- (Optional) Camera/webcam to run the QR scanner. You can also use your phone's camera with the help of DroidCam software.
